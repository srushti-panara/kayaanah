# 📖 About

Kāyaanah AI is a modern dermatology screening platform designed to assist users through AI-powered skin image analysis.

The current implementation focuses on **Psoriasis Detection**, powered by a hybrid **EfficientNet-B4 + Transformer** architecture developed as part of our research work.

The platform combines a clean, accessible interface with intelligent image analysis to provide fast and user-friendly preliminary screening.

> **Disclaimer:** This application is intended for research and educational purposes only. It is **not** a substitute for professional medical advice or diagnosis.

# 🧠 Model Overview

The prediction model combines:

- EfficientNet-B4 for deep feature extraction
- Transformer Encoder for contextual feature learning
- Binary Classification using Sigmoid Activation
- TensorFlow/Keras implementation
- GPU training using Kaggle T4 GPU

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/kayaanah.git
```

Navigate into the project

```bash
cd kayaanah
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open

```
http://localhost:3000
```
